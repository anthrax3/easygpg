# EasyGPG
Zenity GUI to manage GPG keys

![EasyGPG](https://cloud.githubusercontent.com/assets/41558/11421831/600ea7c8-9437-11e5-8468-c9a2bd471e89.png "Zenity GUI to manage GPG keys")

## Install
```
$ sudo chmod +x easygpg
$ ./easygpg
```
