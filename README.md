# EasyGPG
Zenity GUI to manage GPG keys

## What you can do

1. Encrypt file
2. Decrypt file
3. Show file details
4. List public keys
5. List secret keys

## Install
```
$ sudo chmod +x easygpg
$ ./easygpg
```
